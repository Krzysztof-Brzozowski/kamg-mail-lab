# Deployment Checklist

## Phase 1 - Preparation

* [x] GitHub repository created
* [x] Initial documentation created
* [ ] Docker host prepared
* [ ] Storage location selected
* [ ] Backup strategy documented

## Phase 2 - Internet Upgrade

* [ ] Static IP assigned
* [ ] Router installed
* [ ] Port 25 availability confirmed
* [ ] Reverse DNS (PTR) available

## Phase 3 - Mail Server Deployment

* [ ] Stalwart deployed
* [ ] Administrator account created
* [ ] SSL certificate installed
* [ ] Web interface accessible

## Phase 4 - DNS Configuration

* [ ] A record configured
* [ ] MX record configured
* [ ] SPF record configured
* [ ] DKIM record configured
* [ ] DMARC record configured

## Phase 5 - Testing

* [ ] Internal mail flow tested
* [ ] Gmail delivery tested
* [ ] Outlook delivery tested
* [ ] Spam score verified

## Phase 6 - Production

* [ ] Primary mailbox created
* [ ] Backup jobs configured
* [ ] Monitoring configured
* [ ] Documentation updated
