# Backup Strategy

## Storage Layout

### NVMe SSD

Operating System

### SSD Cache

Docker applications

### HDD Storage

Mail data

### USB Backup Drive

Offline backups

## Backup Schedule

### Daily

Mail data snapshots

### Weekly

Full mail backup

### Monthly

Backup verification and restore testing

## Recovery Objectives

* Restore individual mailbox
* Restore complete mail server
* Restore DNS configuration documentation
