# Network Ports

## SMTP

Port 25

Purpose:

* Mail transfer between mail servers

Required:

* Internet inbound
* Internet outbound

## Submission

Port 587

Purpose:

* Authenticated mail sending from clients

Required:

* Internet inbound

## IMAPS

Port 993

Purpose:

* Secure email access

Required:

* Internet inbound

## HTTPS

Port 443

Purpose:

* Web administration
* Webmail access

Required:

* Internet inbound

## HTTP

Port 80

Purpose:

* Let's Encrypt certificate validation

Required:

* Internet inbound
